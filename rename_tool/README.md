# Maya 選択対象リネームツール

・マヤで選択したオブジェクトを、プレフィックス付きで一括でリネームすることができるツール
・よく使うプレフィックスはJSONファイルで保存・呼び出しも可能。

## 機能
- 選択したオブジェクトを一括でリネーム
- 連番で一括管理
- 自分で作ったプレフィックスを頭に付属
- プリセットにプリフィックスを保存
- プリセットから呼び出して反映
## 技術的に工夫した点
- クラスを機能で２つ（リネーム処理とプリセット管理）に分けて、単一責任の原則を徹底。
- プリセットはJSONファイルとして作るため、再起動してもリセットされなくした。
- 多くの作業で使われるプレフィックスはほぼ同じであると考えたため、プリセット機能を作ったことでよりユーザーライクな設計を意識
 
## 使い方
1.スクリプトエディタ（Python）でコードを実行するとツールウィンドウが開く
2.リネームしたいオブジェクトを選択
3.プレフィックスを選択、もしくはプリセットから選択
4.リネームボタンを押すと連番付きでリネームされる

## 開発環境
- Maya 2027
- Maya標準のPython
